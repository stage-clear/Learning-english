## <a id="s1">#</a> 能動態と受動態

```txt
This poem was written by Tom.
この詩はトムによって書かれた
```

<能動態> 「Tさんは君を愛している」  
<受動態> 「君はTさんに愛されている」

### Point: 「能動態」と「受動態」

```txt
受動態: be 動詞 + 過去分詞 + by ~ 「~によって...される」

           (主語)              (動詞)           (目的語)
<能動態>     He      /         wrote        / this letter
<受動態> This letter /      was written     /   by him.
           (主語)     (be 動詞 + 過去分詞)    (by + 目的格)
# 文頭の単語が文末へ、文末の単語が文頭へ、真ん中の1語の動詞が「be 動詞 + 過去分詞」に
```

```txt
例) 次の例を「受動態」に言い換えなさい
1. Everybody loves him. (誰もば彼を愛している)
2. She wrote this novel. (彼女がこの小説を書いた)

回答)
1. He is loved by everybody.
2. This novel was written by her.
```

- 受動態に言いかえるためにはまず、__「目的語」を見つける__ こと
- __目的語は、「~を」と「~に」のどちらかに訳せる語__ を見つければいい
  (もし、目的語がなければ、その文は受動態にできない)
- __目的語を主語にし、能動態の動詞を「be 動詞 + 過去分詞」にかえて、最後に「by + 受動態の主語」をくっつければ受動態の完成__


### Point: 「能動態」→「受動態」への書き換え

```txt
1. 能動態の「目的語」を「主語」にする（目的格→主格にする）
2. 時制や助動詞の有無を確認する
  - 現在の場合 → is[am/are] + 過去分詞
  - 過去の場合 → was[were] + 過去分詞
  - 助動詞の場合 → will[can / may / must など] be + 過去分詞
3. 「by + 能動態の主語」にする(主格 → 目的格にする)
```

能動態の否定文や疑問文を受動態にかえるときは、(慣れるまでは) いきなり受動態にせず、
まず肯定文に直してから受動態にし、それから否定文や疑問文にしたほうが間違えなくていい。


## <a id="s2">#</a> 受動態の注意点

__by + 目的格が省略される場合__

```txt
English is spoken in America.
英語はアメリカで話されている。
```

```txt
例) 次の1~3を受動態に言いかえなさい
1. John will do the work.
2. We can see lions in the zoo.
3. They sell sugar at that shop.

回答)
1. The work will be done by John.
2. × Lions can be seen in the zoo by us.
   ◯ Lions can be seen in the zoo.
3. × Sugar is sold at that shop by them.
   ◯ Sugar is sold at that shop.

# We が「一般の人々」 They が「お店の人」を表してるから by 以下が省略される
# 前後関係から行為者が推測できたり、逆に行為者がわからない場合も by ~ は付けない
```

__「助動詞」「進行形」「完了形」の受動態__

```txt
Mt. Fuji can be seen from my house.
私の家から富士山が見える。
```

```txt
例)
1. We can see Mt. Fuji from here. <能動態>
   → My Fuji can be seen from here. <受動態>

2. They are discussing the problem now. <能動態>
   → The problem is being discussed now. <受動態>

3. Cathy has washed these dishes. <能動態>
   → These dishes have been washed by Cathy. <受動態>

# 助動詞の文は原形
# 進行形の文は ing 形
# 完了形の文は過去分詞
```


### Point: 「助動詞」の受動態

```txt
助動詞の受動態: 助動詞 + be + 過去分詞
```

### Point: 「進行形」の受動態

```txt
進行形の受動態: be 動詞 + being + 過去分詞
```

### Point: 「完了形」の受動態

```txt
完了形の受動態: have[has/had] + been + 過去分詞
```

## <a id="s3">#</a> 第4文型・第5文型の受動態

__第4文型の受動態__

```txt
I was given this book by him.
私は彼にこの本をもらった。
```

第4文型には __目的語が2つ__ あり、__それぞれの目的語を主語にして、2通りの受動態が作れます__ 。


### Point: 「第4文型」の受動態(2つの目的語)

```txt
<能動態>    He     /  gave     /   me  / this book /
<受動態>    I      / was given /       / this book / by him.
<受動態> This book / was given / to me /           / by him.

上: (彼は私にこの本をくれた)
中: (私は彼にこの本をもらった)
下: (この本は彼から私に贈られた)
```

物(直接目的語)が主語になるときは、人(関節目的語)の前に前置詞(to か for)がくる
([to と for のどちらを使うか](01/README.md#s7)は動詞によって決まる)

- __buy, make, cook, sell, read, write などは、ふつう「人」を主語にした受動態を作れない__

```txt
例) 
<能動態> Mary made her sister a doll. (メアリーは妹に人形を作ってやった)
<受動態> ○ A doll was made for her sister by Mary.
         × Her sister was made ~.
```

__第5文型の受動態__

```txt
The dog is called Chibi by him.
その犬は彼にチビと呼ばれる。
```

__第5文型__ で注意するところはただ1つ。__「補語」が名詞の場合、目的語と勘違いして、これを主語にしないこと__ です。

### Point: 「第5文型」の受動態(目的語 + 補語)

```txt
                               (目的語)  (補語)
<能動態>    He   /   calls   / the dog / Chibi.
<受動態> The dog / is called /         / Chibi / by him.

上: (彼はその犬をチビと呼ぶ)
下: (その犬は彼にチビと呼ばれる)
```

補語が形容詞の場合、名詞と同じで「be 動詞 + 過去分詞」の後に置くだけです。

```txt
例)

<能動態>   Bob   /  painted    / the box / white.
<受動態> The box / was painted /         / white / by Bob.

上: (ボブはその箱を白く塗った)
下: (その箱はボブに白く塗られた)
```

## <a id="s4">#</a> 動詞句の受動態

```txt
I was laughed at by everybody.
私はみんなに笑われた。
```

__「動詞句」__ を含む文は受動態にするときに特に気をつけます。

### Point: 「動詞句」の受動態

```txt
<能動態> Everybody /   laughed at   / me.
<受動態>     I     / was laughed at / by everybody.

上: (誰もが私を笑った)
下: (私はみんなに笑われた)
× I was laughed by everybody (at が抜けている)
× I was laughed at everybody (by が抜けている)
```

__動詞句はバラバラにしないで、「1つの動詞(単語)」と同じように扱う__ のが正しい。


## <a id="s5">#</a> 知覚・使役動詞の受動態

```txt
She was seen to go out by me.
彼女は私に出かけるのを見れらた。
```

```txt
例) 次の文を受動態にしなさい
1. I saw her enter the room.
2. He made me go there.

回答)
1. × She was seen enter the room by me.
   ○ She was seen to enter the room by me.
2. × I was made go there by him.
   ○ I was made to go there by him.

```

- __「知覚動詞・使役動詞 + 目的語 + 動詞の原型」の形は、受動態にするときには、動詞の原形に to を付けなければいけない__
- __「動詞」は、1つの文の中に2つ並べられない__ 。並べる時には2つ目の動詞の「頭」か「お尻」に何かをくっつける


## <a id="s6">#</a> by 以外の前置詞を使う受動態

```txt
I'm interested in J-pop
私は J-pop に興味がある
```

### Point: by 以外の前置詞を使う受動態

```txt
1. She is known to everybody.
   (彼女はみんなに知られている)
2. He was caught in shower.
   (彼はにわか雨にあった)
3. We were surprised at the news.
   (私たちはその知らせに驚いた)
4. My parents were disappointed at my test result.
   (両親は私のテストの結果に失望した)
5. The mountains are covered with snow.
   (山々は雪に覆われている)
6. He was pleased with the gift.
   (彼はその贈り物を喜んだ)
7. I am satisfied with my room.
   (私は自分の部屋に満足している)
```

そのほかに、 __能動態の形で受動態の意味を表す動詞もある__ 。


### Point: 受動的意味を持つ動詞・受動態で使われる動詞

```txt
受動的意味を持つ「自動詞」

1. This book sells well. (この本はよく売れる)
2. This knife cuts well. (このナイフはよく切れる)
3. This shirt washes well. (このシャツは選択がよくきく)

受動態の形で使われる「他動詞」

- be shocked (ショックを受ける)
- be hit (ぶつかる)
- be hurt[injured] (けがをする)
- be delayed (遅れる)
- be offended (腹を立てる)
```
