# 時制

## まとめ

1. [過去形](#s1) - be動詞, 一般動詞
2. [未来形](#s2) - 「主語 + will + 動詞の原形~」
3. [進行形](#s3) - 現在進行形, 過去進行形, 未来進行形
4. [完了形](#s4) - 継続, 経験, 完了, 結果, 大過去
5. [完了進行形](#s5) - 現在完了進行形, 過去完了進行形, 未来完了進行形
6. [時・条件を表す副詞節](#s6) - when, before, after, if, unless


## <a id="s1">#</a> 過去形

__be 動詞の過去形__

```txt
He was busy yesterday.
彼は昨日忙しかった。
```

### Point: be 動詞の過去形

- am, is の過去形 - `was` (主語が1人称・3人称の単数) 例) I __was__ sleepy that time.
- are の過去形 - `were` (主語が2人称と複数) 例) They __were__ kind to me.

__一般動詞の過去形__

```txt
I met her last week.
私は先週彼女にあった。
```

### Point: 規則動詞の過去形「ed」のつけ方

```txt
1. [動詞] の語尾 + ed - 例) talk → talk __ed__
2. [e] の語尾 + d - 例) live → live __d__
3. [子音字 + y] の語尾 → y を i　に変えて + ed - 例) study → stud __ies__
4. [短母音 + 子音字] の語尾 → 子音字を重ねて + ed - 例) stop → stop __ped__
```

### Point: 不規則動詞の変化

```txt
         | 原形 - 過去形 - 過去分詞
1. A-A-A : cut - cut - cut
2. A-B-A : come - came - come
3. A-B-B : build - build - built
         : say - said - said
         : sell - sold - sold
         : teach - taught -taught
         : meet - met - met
         : keep - kept - kept
         : make - made - made
4. A-B-C : begin - began - begun
         : get - got - gotten[got]
         : dirve - drove - driben
         : break - broke - broken
         : be[am/is/are] - was/were - been
```

## <a id="s2">#</a> 未来形

__未来形の will と be going to__

```txt
We will go to England next year.
私たちは、来年イギリスに行くつもりだ。
```

#### Point: 「未来形」 = 主語 + will + 動詞の原形

1. 単純未来 - 「〜だろう」 例) It __will__ rain.
2. 意思未来 - 「〜するつもりだ」 例) I __will__ visit my grandmother.

__shall__ は __shall I ~?__ と __shall we ~ ?__ の2つの疑問形だけを覚えておけばいい


#### Point: be going to = will

- 肯定文 - I __will__ play baseball. = I __am going to__ play baseball.
- 否定文 - He __won't__ play baseball. = He __isn't going to__ play baseball.
- 疑問文 - __Will__ you play baseball? = __Are__ you __going to__ baseball?


## <a href="s3">#</a> 進行形

__現在進行形__

```txt
We are walking along the river.
私たちは川に沿って散歩しているところだ。
```

#### Point: 「現在進行形」　= be 動詞（現在形） + 動詞の ing 形

- 現在進行形 - am[is, are] + ~ing 「〜している」「〜しているところだ」 例) I __am cleaning__ my room.

#### Point: 動詞の進行形「ing」のつけ方

```txt
1. <動詞>の語尾 + ing / 例) walk → waling
2. <e>の語尾 → e をとって + ing 例) make → making
3. <短母音 + 子音字>の語尾 → 子音字を重ねて + ing 例) run → running
4. <ie>の語尾 → ie を y にかえて + ing 例) die → dying
```

進行形にできない動詞: know, like, have などの __「状態を表す動詞」__
ただし have が「食べる」という意味のときには進行形も使われる


__過去進行形・未来進行形__

```txt
When she came in, I was watching TV.
彼女が入って来たとき、私はテレビを見ていた。
```

#### Point: 「過去進行形」と「未来進行形」

- 過去進行形 - was[were] + ~ing 「~していた」 例) It __was raining__ when my father came back home.
- 未来進行形 - will be + ~ing 「~しているだろう」 例) He will be studying at this time tomorrow.

__過去進行形__ も __未来進行形__ どちらも `when my father came back home` `at this time tomorrow` など、基準となる時を表す語句が文中に示されていることが多い。


## <a id="s4">#</a> 完了形

__「継続」を表す現在完了(中3)__

```txt
I have lived in Tokyo since 1998.
私は1998年から東京に住んでいる。
```

### Point: 「現在完了」の基本形

```txt
肯定文  have[has] + 過去分詞 ~
否定文  have[has] + not + 過去分詞 ~ 
疑問文  Have[Has] + 主語 + 過去分詞 ~ ?
  Yes, 主語 + have[has].
  No, 主語 + haven't[hasn't].
```

__現在完了には4つの使い方__ がある。


### Point: 現在完了の「継続」

```txt
継続: 過去のあるときから現在まで、同じ状態が続いていることを表す
     「(今まで)ずっと~している[である]」
     「(今まで)~し続けいている」

<過去> He was busy yesterday. (彼は昨日忙しかった)
<現在> He is still busy now. (彼は今もまだ忙しい)
<現在完了> He has been busy since yesterday. (彼は昨日からずっと忙しい)

__「継続」でよく使う語句__

- since (~以来)
- for (~の間)
- How long ~ ? (どのくらいの期間~か?)

昨日の状態が現在まで続いていることを表している。そして、視点はあくまで「現在」に置いている。
「継続」の文は、「昨日から今日」という「線」を表す。
```

__「経験」を表す現在完了(中3)__

```txt
Have you ever seen the movie?
あなたは今までにその映画を見たことがありますか?
```

### Point: 現在完了の「経験」

```txt
経験: 過去から現在までの経験を表す
     「(今までに) ~したことがある」

1. have[has] been to ~ 「~に行ったことがある」
   例) I have been to America.
2. Have[has] + 主語 + ever + 過去分詞 ~?
   例) Have you ever read the book?
3. How many times ~ ? 「何回 ~ したことがありますか?」
   例) How many times have you ever been to America?

__「経験」でよく使う語句__

- ever (今までに, かつて)
- never (1度も ~ ない)
- before (以前)
- often (しばしば)
- once (1回)
- twice (2回)
- ~ times (~回)
```

__「完了」を表す現在完了(中3)__

```txt
The class has already started.
授業はもうはじまっていた。
```

### Point: 現在完了の「完了」

```txt
完了: 動作が今、完了したことを表す
     「(今) ~したところだ」「~してしまった」

例) I have just finished my work.
    I have already finished my work.
    Have you finished your work yet?
    I haven't finished my work yet.

__「完了」でよく使う語__

- just (ちょうど)
- already (もう)
- yet (もう, まだ)
```

### Point: just /already / yet などの位置

1. just, already, never - have[has] と過去分詞の間
  - have[has] just + 過去分詞
  - have[has] already + 過去分詞
  - have[has] never + 過去分詞
2. yet - 文末(疑問文・否定文で使われる)
  - Have[Has] + 主語 + 過去分詞 ~ + yet?
  - 主語 + have[has] + not + 過去分詞 ~ + yet?

already や just は haven't の not と同じ位置に置く。

__「結果」を表す現在完了__

```txt
Winter has come.
冬が来た。
```

### Point: 現在完了の「結果」

```txt
結果: 過去の動作の結果が現在も残っていることを表す
     「~してしまって(その結果、今は)~である」
     「~してしまった」「~になった」「~した」

<過去> He went to Kyoto. (彼は京都に行った) [過去の事実]
<現在> He isn't here now. (彼は今ここにはいない) [現在の状態]
<現在完了> He has gone to Kyoto. (彼は京都へいってしまった)(そして、今ここにいない)

例:
1. I have became a pilot.
2. I have lost my wallet.
3. Tom has come to my home.
```

__現在完了で使えない語句(中3)__

```txt
I met her at school just now.
私はたった今、学校で彼女に会った
```

__現在完了は、はっきりと過去を表す語句とは一緒に使えない__

### Point: 現在完了で使えない語句

- when (いつ) 
  - when と聞かれたら yesterday や last week と答える。つまり __when は特定の日時を聞いている__
- yesterday (昨日)
- ~ ago (~前)
- last ~ (この前~)
- just now (たった今)

```txt
○ I bought this book yesterday.
× I have bought this book yesterday.
```

__過去完了(高校)__

```txt
I had finished my work by dinner
私は夕食までに仕事を終えていた。
```

__「過去完了」__ は現在完了と同じく、__「継続」「経験」「完了」「結果」の4用法__がある。
__現在完了を過去に平行移動させたものが「過去完了」__ と考えればよい。
__「過去完了」には「過去のある時点よりさらに以前に起こった動作・状態を表す(大過去)」用法がある__ 

### Point: 過去完了

```txt
継続   「(過去のある時まで)ずっと ~ していた」
       例) He had lived in Tokyo for 2years when I met him.
経験   「(過去のある時までに) ~ したことがあった」
       例) I knew him because I had met him once.
完了   「(過去のある時には) ~ していた」
       例) She had already left when I got there.
結果   「(過去のある時に) ~ してしまった」
       例) I bought a new watch because I had lost my old one.
大過去 「(過去のある時から見たさらに過去)」
       例) He lent me the video which he had bought.

<継続>   彼は東京に住んでいた  → 彼に会った
<経験>   彼に1度会っていた     → 彼を知っていた
<完了>   彼女は出かけていた    → そこに着いた
<結果>   古い時計をなくした    → 新しい時計を買った
<大過去> 彼はビデオを買った   → 私に貸してくれた
```

__未来完了(高校)__

```txt
I will have finised my work by five.
私は5時までには仕事を終えているだろう。
```

__「未来完了」__ は、__未来の1時点までの「継続」「経験」「完了」「結果」を表す__ 。
__現在完了をそのまま未来に平行移動させたもの__ と考えればよい。

### Point: 3つの「完了形」の違い

- 未来完了 will have + 過去分詞
- 現在完了 have + 過去分詞
- 過去完了 had + 過去分詞

未来完了は口語ではあまり使われず、わざわざ未来完了を使わなくても未来形で十分表現できる。

```txt
<未来完了> The concert will have been over by nine.
<未来形>   The concert will be over by nine.
```


### Point: 未来完了

```txt
継続 「(未来のある時まで) ~し続けているだろう」
     「...で ~ していることになる」
     例) I will have lived in Tokyo for 2 years by next month.
経験 「(未来のある時までに) = を経験しているだろう」
     「もし...すれば~したことになる」
     例) I will have read the book ten times if I read it again.
完了 「(未来のある時までには) ~しているだろう」
 ・  例) I will have watched the DVD by noon.
結果

* 完了と結果ははっきり区別できない場合も多い
```

## <a id="s5">#</a> 完了進行形

__完了形と完了進行形(高校)__

```txt
I have been reading for hours.
私は何時間もずっと読書している。
```

完了形が __「継続」の場合で「動作を表す動詞」のときだけ、「完了進行形」を作ることができる__
（wait や study など、状態を表す動詞以外）

### Point: 「完了形」と「完了進行形」

```txt
            完了形の4つの用法
                   |
         |             |      |     |
        継続          経験   完了  結果
    |           |
動詞(状態)   動詞(動作)
    ↓            ↓
 完了形      完了進行形
```

__「進行形」は、「現在・過去・未来のある時点における進行中の動作」__ を表します。
__「進行形」は「点」__ を表し、__「完了進行形」は「線」を表す__ とも言えます。

### Point: 「現在」「過去」「未来」の完了進行形

```txt
現在完了進行形  have[has] been + ~ing
                「(現在まで)ずっと~している」
                例) I have been reading books for 2 hours.
過去完了進行形  had been + ~ing
                「(過去のある時点まで)ずっと~していた」
                例) I had been reading books for 2 hours when he came in.
未来完了進行形  will have been + ~ing 
                「(未来のある時点で)ずっと~していることになる」
                例) I will have been reading books for 2 hours by five.
```

## <a id="s6">#</a> 時・条件を表す副詞節

__時・条件の副詞節__

```txt
We will start when she comes.
彼女が来れば、私たちは出発するつもりだ。
```

__「時または条件を表す副詞節の中では未来時制は使えない」__ つまり、__「時・条件を表す副詞節の中では、未来のことでも現在時制を使う」

```txt
節 - 文中において <主語 + 述語動詞> の形を備えている語群
句 - <主語 + 述語動詞> の形を備えていない語群
     例) "in the garden" のような「前置詞 + (代)名詞」が句の代表的な形

副詞節 - 副詞の働き(動詞や形容詞を装飾)をする節
         (「装飾する」というのは、動詞や形容詞などを説明して、意味をはっきりさせる役割を果たすこと)
```

(メモ) 目的語には名詞が使われる → 名詞節


### Point: 時を表す副詞節 (when)

```txt
時を表す副詞説の中では、未来のことでも現在時制を使う
例) I will tell you when she comes back <副詞節>
            ↑___________|    (× will come)
    (彼女が帰ってきたら、私はあなたに知らせます)

    Do you know when she will come next time ? <名詞節>
                         (× comes)
    (彼女が今度いつ来るかわかりますか?)
    = 彼女が今度いつ来るかということ [を] 知っていますか?
```

### Point: 条件を表す副詞節 (if)

```txt
条件を表す副詞節の中では、未来のことでも現在時制を使う

例) I will stay at home if it rains tomorrow. <副詞節>
            ↑____________|    (× will rain)
    (もし明日雨が降れば、私は家にいるつもりだ)
    
    I don't know if it will rain tomorrow.
                       (× rains)
    (明日雨が降るかどうか私にはわからない)
    = 明日雨が降るかどうかということ [を] 私は知らない
```

### Point: 「時や条件を表す副詞節」で使われる接続詞

```txt
時   - when (~するとき)
     - whenever (~するときはいつでも)
     - as soon as (~するやいなや)
     - before (~する前に)
     - after (~した後で)
     - till[untill] (~するまでずっと)
条件 - if (もし~ならば)
     - unless[if ~ not] (もし~でないならば)

* これら接続詞のうち when と if 以外は「副詞節」しか作らない。
  つまりこれらの接続詞では未来のことは常に現在時制を使う。
```
