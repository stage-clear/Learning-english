## <a id="s1">#</a> 不定詞の名詞用法

__不定詞の名詞用法__

```txt
My dream is to be a movie star.
私の夢は俳優になることだ。
```

__不定詞には3つ用法があるが、「3種類の意味がある」というだけのこと__ 。

### Point: 不定詞: to + 動詞の原形

```txt
1. 名詞用法 (=名詞と同じ働き)「~すること」
   → 名詞用法の不定詞は、「主語」「目的語」「補語」の3つの働きをする
2. 副詞用法 (=副詞と同じ働き)「~するために、~して」
   → 動詞や形容詞を修飾する
3. 形容詞用法 (=形容詞と同じ働き)「~するための、~すべき
```

### Point: 名詞用法の不定詞

```txt
1. 主語になる「~することは」
   例) English is easy. (英語はやさしい)
       To study English is easy. (英語を勉強することはやさしい)

2. 目的語になる「~することを」
   例) I like English. (私は英語が好きだ)
       I like to study English. (私は英語を勉強するのが好きだ → することを好む)

3. 補語になる「~することだ」
   例) He is a teacher. (彼は先生だ)
       My dream is to be a teacher. (私の夢は先生になることだ)

# 名詞を置くべき「主語・目的語・補語」のところに、「to + 動詞の原形」が入っている
```

## <a id="s2">#</a> 不定詞の副詞用法

__不定詞の副詞用法(1)__

```txt
I went to the store to buy some sweets.
私はお菓子を買うためにその店に行った。
```

### Point: 副詞用法の不定詞

```txt
1. 目的を表す「~するために」
   例) I came here to learn dance. (動詞を修飾)
   (私はダンスを習うためにここへ来た)
   * go, come などの自動詞がよく使われる

2. 原因・理由を表す「~して」
   例) I was surprised to hear the news. (形容詞・過去分詞を修飾)
   (私はそのニュースを聞いて驚いた)
   * be glad[happy] to ~ (~してうれしい)、be surprised to ~(~して驚く)などの感情を表す形容詞や過去分詞がよく使われる
```

### Point: 「結果」を表す副詞用法の不定詞

```txt
1. awoke[woke up] to find ~「目が覚めてみると~とわかった」
   例) I awoked to find myself on the bed.
   (目が覚めると、私はベッドの上にいた)

2. grew up to be ~ 「成長して~になった」
   例) He grew up to be a lawyer.
   (彼は成長して弁護士になった)

3. lived to be ~ 「~歳まで生きた」
   例) She lived to be ninety-four. 
   (彼女は94歳まで生きた)

4. 動詞の過去形... + never to + 動詞の原形 「...して、2度と~しなかった」
   例) He left here never to return.
   (彼はここを去って、2度と戻らなかった)

5. 動詞の過去形... + only to + 動詞の原形 「...したが、結局~に終わった」
   例) He studied hard only to fail.
   (彼は熱心に勉強したが、結局失敗に終わった)

# 目的を表す副詞用法「~するために」のように、後ろから前に意味をとってしまわないように気をつける。
```

__不定詞の副詞用法(2)__

```txt
He worked hard in order to buy a house.
彼は家を買うために熱心に働いた。
```

__目的の意味をはっきりさせたい場合には、他の意味と区別するために、to の前に in order か so as をつけることがある__ 。

### Point: 「目的」を表す副詞用法の不定詞

```txt
to
in odrder to  + 動詞の原形「~するために」
so as to
```

- __不定詞を否定する not は、to の直前に置く__ 
- in order to や so as to の場合、「in order not to + 動詞の原形」「so as not to + 動詞の原形」になる


### Point: 不定詞の否定形

```txt
不定詞の否定形: 「~しないため[よう]」

not to + 動詞の原形

例) I told him to go there.
    (私は彼にそこへ行くように言った)
    → I told him not to go there.
    (私は彼にそこへ行かないように言った)

例) I decided not to see the movie.
    (私はその映画を見ないと決めた)

例) Be careful not to be loud.
    (うるさくならないように注意しなさい)
```

__「原形不定詞 (to のつかない不定詞)」というのもある__ 。
「使役動詞 + 目的語 + 動詞の原形「知覚動詞 + 目的語 + 動詞の原形」や、
to 不定詞を使った「動詞 + 目的語 + to + 動詞の原形」といった形については、第5文型を参照。


## <a id="s3">#</a> 不定詞の形容詞用法

__不定詞の形容詞用法__

```txt
I want something to eat.
私は何か食べ物がほしい。
```

### Point: 形容詞用法の不定詞

```txt
名詞・代名詞の直後に置かれ、その名詞・代名詞を修飾する「~する(ための)...」

例) I want something to eat.
    (私はなにか食べるもの(食べるためのなにか)がほしい)

例) I have time to play tennis.
    (私はテニスをする時間(するための時間)がある)

例) He has no house to live in.
    (彼は住む家(中に住むための家)がない)
```

形容詞は名詞の前に置くのがふつうだが、名詞の後におく場合がある。__「~thisng + 形容詞」__ 。
__形容詞用法の不定詞は、名詞・代名詞の直後に置かれた「to + 動詞の原形」が、あとから名詞・代名詞を修飾する__
book, house, time, money のような単語のときによく出てくる。

```txt
例)
1. a house to live in ← live in a house (家に住む)
   (住む家 = 中に住むための家)

2. a chair to sit on ← sit on a chair. (椅子に座る)
   (座る椅子 = 上に座るための椅子)

3. a friend to play with ← play with a friends (友達と遊ぶ)
   (遊び友達)

4. a friend to talk with ←　talk with a friend (友達話し合う)
   (話し合える友達)

5. a pencil to write with ← talk with ← play with a friends (鉛筆で書く)
   (書く鉛筆)

6. money to buy the book with. ← buy the book with money
```

## <a id="s4">#</a> いろいろな不定詞の形

__疑問詞+ 不定詞__

```english
I don't know where to go.
私はどこへ行ったらよいかわからない
```

__「疑問詞 + to ~」__ は、know や learn の目的語となることが多い。

### Point: 疑問詞 + 不定詞

```english
1. how to ~ 「~のしかた」
   例) I learned how to make a cake.
   (私はケーキの作り方を習った)

2. what to ~ 「何を~したらよいか」
   例) I don't know what to do.
   (私は何をしたらよいかわからない)

3. which to ~ 「どちらを~したらよいか」
   例) She didn't know which do select.
   (彼女はどちらを選んだらよいかわからなかった)

4. when to ~ 「いつ~したらよいか」
   例) Please tell me when to go there.
   (いつそこへ行けばよいか教えて下さい)

5. where to ~ 「どこへ~したらよいか」
   例) He didn't know where to go.
   (彼はどこへ行ったらよいかわからなかった)
```

__be to + 動詞の原形__

```english
We are to meet at seven.
私たちは7時に会うことになっている。
```

be の後に不定詞が続いて __be to の形になると、「予定」「義務」「可能」「運命」「意図・目的」の5つの意味を表す__ 。

### Point: be to + 動詞の原形

```english
1. 予定「~することになっている、~する予定だ」
   例) I am to go to his house. (私は彼の家に行く予定だ)

2. 義務「~すべきだ」
   例) You are to leave at once. (君はすぐに出発すべきだ)

3. 可能「~することができる」
   例) No clouds were to be seen in the sky. (空には雲が1つも見えなかった)
   * 否定・疑問で「be to be + 過去分詞」の形で使う

4. 運命「~する運命だ」
   例) He was never to see his hometown again. (彼は二度と故郷に帰れない運命だった)

5. 意図・目的「~するつもりだ」
   例) If you are to successd, you should study harder. (もし成功するつもりなら、君はより熱心に勉強するべきだ)
   * if節で用いる
```

## <a id="s5">#</a> 不定詞の主語・時制、独立不定詞

__不定詞の意味上の主語__

```english
It is diffcult for us to speak english.
私たちが英語を話すのは難しい
```

### Point: 不定詞の「意味上の主語」

```english
1. 文の「主語」と一致する場合
   例) I want to buy a CD. (私はCDを買いたい)
       → I buy (私が買う)

2. 文の「目的語」と一致する場合
   例) I want you to buy a CD. (私はあなたにCDを買ってほしい)
       → You buy (あなたが買う)

3. 「for + 人」で表す場合
   例) It is easy for me to speak English. (英語を話すのは私には簡単だ)
       → I speak. (私が話す)

4. 「of + 人」で表す場合
   例) It is kind of you to help us. (手伝ってくださってありがとう)
       → You help (あなたが手伝う)
```

__for と of の使い分けは、「人の性格」を表す形容詞の後には of がくると覚える__ 。


__完了形の不定詞__

```english
His brother seems to have been ill.
彼の弟は病気だったらしい。
```

### Point: 不定詞の時制

```english
1. He seems to be ill. = It seems that he is ill.
   (彼は病気らしい)         ^^^^^         ^^ → 時制が一致

2. He seems to have been ill. = It seems that he was ill.
   (彼は病気だったらしい)          ^^^^^         ^^^ → 時制のずれ
  * 右の was は文脈によって has been になることもある

3. He seemed to be ill. = It seemed that he was ill.
   (彼は病期らしかった)      ^^^^^^         ^^^ → 時制が一致

4. He seemed to have been ill. = it seemed that he has been ill.
   (彼は病期だったらしかった)       ^^^^^^         ^^^^^^^^ → 時制のずれ
```

不定詞は、それ自体の時制を持たないので、述語動詞の時制に一致する。
(2)や(4)のように時制にずれがある場合、「to + 動詞の原形」の形は使えないので、
述語動詞の時制よりも「以前」の出来事を表す場合は、完了形の不定詞「have to + 過去分詞」の形を使う。

### Point: 過去に実現しなかった希望・意図など

```english
過去に実現しなかった希望・意図を表す不定詞:
「~するつもりだったが、できなかった」

     hoped, wishied, wanted,
人 + expected, intended, meant, + to have + 過去分詞
     was[were]

例) I intended to have visited my grandfather.
    = I had intended to visit my grandfather.
    = I intended to visit my grandfather, but I couldn't.
    (私は祖父を訪れるつもりだったが、できなかった)
```

__希望・意図などの意味を持つ、上のような動詞の過去形に完了形の不定詞を続けると、その希望や意図が実現しなかったことを表す使い方__ 。
動詞のほうを過去完了にして「to + 動詞の原形」をつけても同じ意味になる。


__その他の不定詞を使った表現__

```english
I'm too sleepy to do my homework.
私はあまりに眠くて宿題ができない。
```

### Point: その他の不定詞を使った表現

```english
1. too + 形容詞[副詞] + to + 動詞の原形
   「あまりに...なので~できない」
   例) I am too tired to run.
       = I am so tired that I can't run.
   (私はあまりに疲れているので走ることができない)

2. how long does it take to + 動詞の原形 ~ ?
   「~するのにどのくらい時間がかかりますか?」
   例) How long does it take to ge to the library?
   (図書館までどのくらい時間がかかりますか?)
```

`too ... to ~` は後ろから前に訳すと「~するにはあまりに...すぎる」となるが、
前から__「あまりに...なので~できない」と訳す__ ほうがふつう。
`so ... that ~` の形で書き換えられる。

### Point: 独立不定詞
不定詞の形で文全体を装飾する句のことをいう。

```english
1. to begin[start] with 「まず第一に」
   例) To begin with, it is well-colored.
   (第一に、それは色がよい)

2. to tell (you) the truth 「実を言うと」
   例) To tell the truth, I didn't get any sleep last night.
   (実を言うと、昨晩一睡もできなかった)

3. to be frank with you 「率直に言えば」
   = frankly speaking
   例) To be frank with you, I don't like her.
   (率直に言えば、私は彼女が好きではない)

4. to be sure 「たしかに」

5. strange to say 「不思議なことに」「妙な話だが」

6. so to speak 「いわば」
   例) She is, so to speak, a walking dictionary.
   (彼女は、いわば生き字引だ)
```

## <a id="s6">#</a> 動名詞

__動名詞の基本__

```english
I enjoy siging songs.
歌を歌うのは楽しい
```

- 動名詞とは、__動詞に「ing」をつけることで名詞と同じ働きをする__ 。
- 動名詞は不定詞ととても仲がいい。
- __「名詞用法の不定詞」のほとんどが動名詞に書き換えられる__ 。
- __不定詞と動名詞がお互いに交換できるのは、「主語」「補語」「目的語」の場合__ だ。

```english
例)
主語の場合
1. To speak English is easy. <不定詞>
   (英語を話すことはやさしい)
2. Speaking  English is easy. <動名詞>
```

どちらも「話す」という単語を入れて文を作りたいが、動詞は主語にならないため `speak` は入れられない。
そこで、:one: は頭に `To` をつけ、:two: はお尻に `ing` をつけることで名詞に変身する。
__どちらも同じ意味だから、「不定詞」と「動名詞」がお互いに交換できる__ 。


```english
例)
1. like の目的語の場合
   ○ I like to study math. <不定詞>
     (私は数学を勉強するのが好きだ)
   ○ I like studying math. <動名詞>

2. want の目的語の場合
   ○ I want to study math. <不定詞> 
     (私は数学を勉強したい)
   × I want studying math. <動名詞>
```

__like は「不定詞」と「動名詞」の両方と仲良くできるが、want は「動名詞」と相性がよくない__ 。
他に、mind, enjoy, finish など。


- __動名詞でもう1つ押さえておきたいのは、「前置詞の目的語になる」__ ということ。
- __前置詞の直後に動詞を置くときは、動詞を動名詞にかえればいい__ 。
- __動名詞の否定形は不定詞と同じで、直前に not を置く__ 。

### Point: 目的語に不定詞・動名詞をとる動詞

```english
1. 動名詞のみをとる動詞: enjoy / finish / mind / give up など
   例) I enjoyed playing soccer. (私はサッカーを楽しんだ)
       She finished writing a letter. (彼女は手紙を書き終えた)
       Do you mind helping me? (私は手伝ってくれませんか?)

2. 不定詞のみをとる動詞: want / wish / hope / decide など
   例) I want to buy a CD player. (私はCDプレーヤーを買いたい)
       I hope to see her again. (私は再び彼女に会いたい)
       He decided to be a teacher. (私は先生になろうと決心した)

3. 動名詞・不定詞どちらもとる動詞: like / love / begin / start など
   例) I like playing [to play] games. (私はゲームをするのが好きだ)
       He began cleaning[to clean] our classroom. (彼は教室を掃除し始めた)
       It started raining[to rain]. (雨が降り始めた)

4. 動名詞と不定詞では意味が異なる動詞: stop / remember / forget
   例) He stopped smoking. (彼はたばこを吸うのをやめた)
       He stopped to smoke. (彼はたばこを吸うために立ち止まった)
   例) I remember seeing him. (私は彼に会ったことを覚えている)
       Remember to mail the letter. (忘れずに手紙を出して)
　 例) I forgot posting the letter. (私は手紙を出したことを忘れていた)
　     Don't forget to mail the letter. (手紙を出すのを忘れないで)
```

__動名詞の意味上の主語・完了形の動名詞__

```english
Do you mind my asking the reason?
理由をお尋ねしてもよろしいでしょうか?
```

__動名詞の意味上の主語__ が、文の主語や一般の人々ではない場合は、__名詞・代名詞の所有格[目的格]を動名詞の直前に置く__ 。

### Point: 動名詞の「意味上の主語」__

```english
名詞・代名詞の所有格[目的格]を動名詞の直前に置く
例) I am sure of his[him] passing the exemination. <動名詞>
    = I am sure that he will pass the examination.
    (私は彼がきっと試験に合格すると思う)
```

__述語動詞の時制よりも「以前」を表す場合は、完了形の動名詞__ を使う。


### Point: 「完了形」の動名詞

```english
述語動詞の時制よりも「以前」の出来事を表す場合は、完了形
の動名詞(having + 過去分詞)の形を使う

例) He denied having done such a thing. <動名詞>
    = He denied that he had done such a thing.
    (彼はそんなことはしなかったと言った)
```

__動名詞を使った慣用表現__

```english
I'm looking forward to working with you.
一緒にお仕事をさせていただくのを楽しみにしています。
```

### Point: 動名詞を含む慣用表現

```english
1. look forward to ~ing (~するのを楽しみにする)
2. object to ~ing (~ するのに反対する)
3. be used[accustomed] to ~ing (~するのに慣れている)
   *used + to + 動詞の原形(以前は~したものだった)と区別する
4. with a view to ~ing (~する目的で)
5. when it comes to ~ing (することとなると)
6. What do you say to ~ing? (~しませんか?)
7. be worth ~ing (~する価値がある)
8. burst out ~ing (突然~しだす)
9. feel like ~iing (~したい気がする)
10. There is no ~ing (~できない)
11. It is no use[good] ~ing (~しても無駄だ)
12. cannot help ~ing (~せざるをえない)
    = cannot but + 動詞の原形
```
